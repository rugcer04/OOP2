Smulkus aprašymas

Abstrakti bazinė klasė žmogaus duomenims saugoti.

Klasė apibrėžia bendras savybes ir metodus, reikalingus žmogaus duomenims apdoroti, negalima sukurti žmogaus tipo objektų, o tik objektus iš jos išvestinių klasių. Klasė apibrėžia pagrindinius atributus (vardas, pavardė). Ji yra abstrakti, nes turi virtualią funkciją, kuri turi būti įgyvendinta išvestinėje klasėse.
